Kairos Launcher

Common society uses smartphone everyday, everywhere and all the time, which often entails productivity problems. This happens not only because of the user but because the smartphone it's not prepared and/or ready to adapt itself to the user's needs.

Kairos Launcher was born to change this. This Android Launcher can recognize the user's context, adjusting the smartphone to his every needs, bloking inappropriate notifications, reorganize the home screen to the context apps and giving the user stats about the time spent per application and context.
